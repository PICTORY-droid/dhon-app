# DH On — 다현건설 현장관리 앱

> 다현건설 현장관리자를 위한 안드로이드 현장관리 시스템

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 📄 출력 일보 | 현장 일보 파일 관리 (PDF, Word, Excel) |
| 📋 서류 관리 | 안전 / 품질 / 노무 / 행정 서류 분류 관리 |
| 📸 현장 사진 | 공정 태그별 현장 사진 촬영 및 관리 |
| 📇 명함 관리 | 명함 촬영 및 카테고리별 연락처 관리 |
| 🔊 음성 번역 | 한국어 ↔ 베트남어 / 중국어 실시간 통역 |
| 📢 공지 전달 | 외국인 근로자 대상 다국어 공지 발송 |
| ⚙️ 날씨 알림 | 현장 지역 기반 다음날 날씨 알림 |

---

## 설치 방법

> 플레이스토어 미등록 앱입니다. APK 파일로 직접 설치합니다.

1. 카카오톡으로 APK 파일 수신
2. APK 파일 실행
3. 설정 → 보안 → 출처를 알 수 없는 앱 설치 허용
4. 설치 완료 후 DH On 아이콘 확인

---

## 기술 스택

- **언어**: Kotlin
- **플랫폼**: Android (minSdk 26 / targetSdk 36)
- **번역**: Google ML Kit (온디바이스)
- **음성 인식**: Google Speech API
- **OCR**: Google ML Kit Text Recognition

---

## 서류 분류 체계

```
서류 관리
├── 안전서류 (TBM일지, 안전교육일지, 위험성평가서 등)
├── 품질서류 (콘크리트 타설일지, 레미콘 납품서 등)
├── 노무서류 (근로계약서, 근로자 명부, 임금대장 등)
└── 행정/계약서류 (하도급 계약서, 기성 청구서류 등)
```

---

## 데이터 저장 방식

- 모든 데이터는 **기기 로컬 저장** (외부 서버 미사용)
- 1개월 경과 파일 → 지난서류 보관함 자동 이동
- 1년 경과 파일 → 휴지통 자동 이동
- 휴지통 1년 경과 → 영구 삭제
